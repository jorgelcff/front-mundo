# README

## Tecnologias

Todas as tecnologias usadas no desenvolvimento do projeto.

 - Angular 16+
 - Typescript
 - Bootstrap 5.2
 - SweetAlert2
 - Ng-Toarst

## Requisitos para desenvolvimento
 - NodeJs LTS 18

## Iniciando a aplicação
### Instalando as dependências
    npm i
### Executando a aplicação local
 - **Desenvolvimento**
<br>
 `ng serve -o`

 - **Stage**
<br>
 `ng serve -c=stage -o`

 - **Produção**
<br>
  `ng serve -c=production -o`
